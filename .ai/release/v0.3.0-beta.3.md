# v0.3.0-beta.3

## Changes since v0.3.0-beta.2

- README now presents Project Copilot as an AI Native Engineering Memory Layer for Codex.
- Added rule-based Memory Governance and Drift Detection on top of validation snapshots.
- Added README Drift Check, ADR Governance, Session Quality, Legacy Migration Report, and Multi-Project Validation.
- Added ADR 0007 to record the governance boundary and why it belongs in derived validation.
- Replaced placeholder session archive content with a real P2 governance summary.

## Governance Scope

- README Drift reports documentation drift, but does not auto-edit README.
- ADR Governance checks numbering, status, and `Superseded By` chains.
- Session Quality checks archive length, noise, and ADR duplication with rule-based heuristics only.
- Legacy Migration Report classifies old files without auto-deleting them.
- Multi-Project Validation reads project sources from `validation/sources.yaml`.
